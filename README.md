# Layout Industriel - Visualisation des Machines

Ce projet transforme vos données Excel en un layout industriel interactif.

## 📋 Structure du projet

```
layout/
├── web/
│   ├── index.html          # Interface web interactive
│   ├── style.css           # Styles
│   └── script.js           # Logique JavaScript
├── data/
│   └── machines_data.json  # Données des machines
└── README.md
```

## 🚀 Utilisation

1. Ouvrez `web/index.html` dans votre navigateur
2. Utilisez les contrôles pour explorer le layout
3. Cliquez sur les machines pour voir les détails

## 🎯 Fonctionnalités

✨ **Visualisation en grille** - Affichage clair de toutes les machines  
🔍 **Zoom interactif** - Zoomez avant/arrière  
🔎 **Recherche** - Trouvez rapidement une machine  
ℹ️ **Panneau d'info** - Détails de la machine sélectionnée  
📱 **Responsive** - Fonctionne sur tous les appareils
